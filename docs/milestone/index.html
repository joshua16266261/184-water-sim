Project Milestone


Accomplishments

Particle Simulation

For particle simulation, we have written working code that can generate simulations that exhibit the expected behavior, such as incompressibility.

To test our simulations, we have been plotting the particle positions in Matplotlib to get animations shown in Preliminary Results.

However, we still need to fine-tune some parameters to get more stable results that also exhibit more surface tension.

Mesh Generation

For fluid particle rasterization with the marching cubes algorithm, we have successfully created a working build that can create a feasible looking 3D mesh based on the distribution of fluid particles. For our preliminary results, we have generated a one frame cube given an equal distribution of fluid particles within our boxed environment. Relative to our plan, we have almost met the expectations, however there still needs to be some fine tuning of the input parameters and some speed optimizations in order to meet our goal of a visually accurate and efficient rendering of fluid particles.

Progress Reflection

Progress has been slower than expected on both fronts. For both simulating and rendering the fluid we are still working on tuning our parameters when we expected to be simulating water-air interactions by now. We've replanned our schedule to fit our current situation and still hope to present a realistic water simulation with water-air interactions by May 5.
